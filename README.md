# Contact Manager App

A simple contact manager application built with HTML, CSS, and JavaScript.

## Features
- Add new contacts with name and phone number
- Store contacts in browser's local storage
- Delete contacts
- Responsive design using Bootstrap

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome Icons
